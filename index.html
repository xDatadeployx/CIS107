How Verse.js Works

Components

Create reusable UI pieces with Verse.component(). Each component manages its own state and renders its own HTML.

State

Reactive state with state: {}. When state changes via setState(), the component automatically re-renders.

Events

Attach events using v-on attributes in your render function. Verse handles binding automatically.

Mount

Mount any component to any DOM element with Verse.mount(component, '#selector').

Example Code

const Counter = Verse.component({
  state: { count: 0 },

  methods: {
    increment() {
      this.setState({ count: this.state.count + 1 });
    },
    decrement() {
      this.setState({ count: this.state.count - 1 });
    }
  },

  render() {
    return `
      <div class="counter">
        <button v-on="decrement">-</button>
        <span>${this.state.count}</span>
        <button v-on="increment">+</button>
      </div>
    `;
  }
});

Verse.mount(Counter, '#counter-app');
